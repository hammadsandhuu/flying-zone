# Project Color Palette

This document contains all colors used in the LRT-Version project.

## CSS Variables (Primary Colors)

Defined in `public/assets/css/style.scss` and `public/assets/css/dashboard.scss`:

```css
:root {
  --white-color: #fff;
  --black-color: #000;
  --title-color: #100C08;
  --text-color: #787878;
  --primary-color1: #0493d7;
  --primary-color1-opc: 4, 147, 215;
  --primary-color2: #3e4a8c;
  --primary-color2-opc: 62, 74, 140;
}
```

## Color Usage Summary

### Most Used Colors
1. **Primary Sky Blue** (`#0493d7`) - Main brand color (ocean/sky theme)
2. **Primary Deep Blue** (`#3e4a8c`) - Secondary brand color (deep ocean/twilight theme)
3. **Title Color** (`#100C08`) - Text headings
4. **Text Color** (`#787878`) - Body text
5. **White** (`#fff`) - Backgrounds, text on dark
6. **Black** (`#000`) - Text, borders
7. **Light Grays** (`#eee`, `#ddd`, `#f3f3f3`) - Borders, backgrounds
8. **Dark Backgrounds** (`#1d231f`, `#0A1019`, `#202f59`) - Section backgrounds

### Color Categories
- **Theme Colors**: Sky Blue (`#0493d7`) and Deep Blue (`#3e4a8c`) - Nature/ocean theme
- **Neutral Grays**: Extensive grayscale from `#000` to `#fff`
- **Accent Colors**: Various blues and nature-inspired colors for UI elements
- **Background Colors**: Light beiges, blues, and dark sections

## Notes

- Colors are primarily defined as CSS variables in `:root`
- Many colors use RGBA for transparency effects
- Some colors are hardcoded in specific components
- The project uses a nature/ocean theme with sky blue and deep blue colors
- Dark mode sections use dark backgrounds like `#1d231f` and `#0A1019`
- **Updated**: Primary colors changed from green/yellow to blue ocean theme (`#0493d7` and `#3e4a8c`)
- **Total unique hex colors: 317**
- Some colors appear in both uppercase and lowercase (e.g., `#100C08` and `#100c08`) - these are functionally the same
- Many Bootstrap and third-party library colors are included in the count
